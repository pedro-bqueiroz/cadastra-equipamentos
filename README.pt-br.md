# cadastra-equipamentos

Um protótipo de sistema de gerenciamento digital projetado para registrar equipamentos em um laboratório de engenharia.
Este programa foi desenvolvido por Pedro Queiroz e Ventura335 para um trabalho de fim de semestre, há algum tempo. 
Os requisitos e objetivos do projeto foram revisados para se adequarem melhor a demandas do dia-a-dia.
Algumas funções são de longe menos eficientes que o ideal, e podem consumir uma quantidade de memória absurda para sistemas embarcados, mas mantemo-nas porque um dos requisitos era usar os conhecimentos que tínhamos obtido ao longo do curso. Para o futuro, pode ser possível integrar esse sistema a um banco de dados, ou criar uma interface gráfica (usando GTK+; não espere que se pareça com algo feito com Qt).

Ao executar o programa pela primeira vez, ou para assumir controle total, escreva "ICE CREAM" ao iniciar a sessão. Você poderá configurar usuários e senhas mais tarde.
