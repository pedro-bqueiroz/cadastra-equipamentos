# cadastra-equipamentos

Un prototipo de sistema de gestión digital diseñado para registrar equipos y informaciones personales en un laboratorio de ingeniería.
Este proyecto fue desarrolado por Pedro Queiroz y Ventura335 como un ejercicio de fin de semestre, hace algun tiempo.
Los objetivos originales del proyecto no eran muy prácticos, así que los mejoramos basándonos en la realidad. Algunas funciones no fueran modificadas, porqué uno de los objetivos originales fue utilizar lo que hemos aprendido. 
Para el futuro, es posible incorporar algunas implementaciones útiles como integración a algún banco de datos, o alguna interfaz gráfica (GUI con GTK+; no esperes que tenga el aspecto de algo hecho con Qt).

Para obtener control total, escribe "ICE CREAM" al iniciar sesión.
